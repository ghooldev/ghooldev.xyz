Downloading Our Software

Getting our software downloaded and installed on your PC, is a very simple process.

Here are the steps to follow:

    1. Go to the Ghool Shop
    2. Find the software that you want to download.
    3. Extract the .zip (if there is one).
    4. Run the EXE!
    PS. If you want the source code, select "View On GitHub"
